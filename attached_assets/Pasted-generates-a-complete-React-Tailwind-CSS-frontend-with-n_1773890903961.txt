1.3 SEO strategy baked into the prompt
Every page has a unique title tag (55–60 chars) and meta description (148–155 chars)
H1 on every page contains the primary keyword
JSON-LD schema: Organization (homepage), FAQPage (alternative pages), Product (pricing)
Open Graph meta tags for social sharing on all pages
Hreflang tags for 7 markets (en, fr, de, es, it, pl, es-AR, x-default)
Semantic HTML: <article>, <section>, <nav>, <main>, <aside> used correctly
All images use descriptive alt text with keywords
robots meta tag: index, follow on all public pages
Canonical URLs on every page
llms.txt file for AI search engines (ChatGPT, Perplexity, Google AI Overviews)
